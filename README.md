Название проекта: «Тыва-Бот: Путеводитель по Кызылу»

Роли:
- Гость
- Авторизованный Пользователь 
- Эксперт
- Модератор
- Администратор

Функции каждого по ролям:

Гость:
- Просмотр главного меню
- Просмотр достопримечательностей по категориям
- Получение карты с метками
- Поиск по названию места через текстовый ввод
- Регистрация в системе

Пользователь:
- Все функции гостя
- Личный кабинет с избранным
- Лайки/дизлайки мест
- Комментарии к местам
- Создание личных маршрутов
- Получение персональных рекомендаций

Эксперт:
- Все функции пользователя
- Добавление новых мест через специальные команды
- Редактирование информации о местах
- Создание тематических маршрутов
- Добавление фото/описаний

Модератор:
- Все функции эксперта
- Модерация пользовательского контента
- Редактирование любых карточек мест
- Управление событиями
- Просмотр статистики

Администратор:
- Полный доступ ко всем функциям
- Управление пользователями и ролями
- Рассылка уведомлений
- Аналитика

Базовые функции:
- Интерактивное меню с кнопками
- Категории достопримечательностей с фото и описанием
- Карта города со статичными метками или ссылкой на карты
- Поиск по названию места через текстовый ввод

Описание запросов к боту в формате запрос-ответ:

- Пользователь отпраляет /start
- Бот отвечает "Добро пожаловать в Кызыл! Выбери действие:"
    [Меню]:
    [Достопримечательности] [Карта города]
    [Поиск] [О городе]
  
- Пользователь нажимает: Достопримечательности
- Бот: "Выбери категорию:"
    [Исторические/Культурные] 
    [Природные] [Кафе и рестораны]
    [Назад]
  
- Пользователь нажимает: Карта города
- Бот: "Карта Кызыла с основными достопримечательностями:"
    [Открыть в Картах] [Назад]

- Пользователь нажимает: Поиск
- Бот: "Введите название места для поиска:"
- Пользователь: музей
- Бот: "Найдено по запросу 'музей':"
    [Кнопки с найденными результатами] [Назад]

- Пользователь нажимает: О городе
- Бот выводит информацию о городе ... 
    [Назад]
